# Changelog

Orbit remains `0.0.1` while its contracts are being proven.

## Unreleased

- Aligned the public experience around exact monitored sources, post-run payoff, source health, partial truth, endpoint/item semantics, and browser-only Outcome JSON export.
- Split clone and package quickstarts and documented tested commands, expected output, local development, hosted constraints, and exact limits.

- Added public JSON, RSS, and GitHub Releases adapters with bounded public-source collection.
- Added deterministic and injected evidence-safe assisted lens contracts.
- Added evidence-linked JSON and Markdown briefs, generated gaps, non-executing review notes, per-source receipts/truncation, and versioned manual outcomes.
- Added a “Collected evidence distribution” lane-by-adapter review without collapsing coverage into an opaque quality score.
- Reframed the webpage around a clear collect → inspect → choose flow; suggestions are explicitly non-executed review notes with copy controls, not a fake approval queue.
- Added CLI, minimal non-mutating MCP subset, `loops.yaml`, skill, Svelte/Hono review UI, package smoke, and CI.
- Hardened request/response streaming limits, strict contract validation, unsafe URL rejection, redirect refusal, CSP/security headers, and hosted exact-spec execution.
- Deployed the public-source reference at [orbit.coey.dev](https://orbit.coey.dev) and dogfooded the same spec through CLI, loops-yaml, hosted HTTP, browser review, and My AX.
